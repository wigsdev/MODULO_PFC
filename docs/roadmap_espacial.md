# Roadmap: Finalización del Módulo Espacial

## Objetivo General
Completar la implementación de todas las sub-secciones del Módulo Espacial, poblando el contenido real de las páginas (actualmente placeholders) e integrando los datasets correspondientes.

## Fases de Implementación

### Fase 1: Planificación Rural Agropecuaria
- [ ] **Frontera Agrícola Nacional**: Implementar visualización de mapa y estadísticas.
- [ ] **Zonificación de Cultivos**: Integrar capas de zonificación.
- [ ] **Zonas Susceptibles**: Implementar mapa de riesgos de mercado.
- [ ] **Áreas de Formalización**: Mostrar áreas priorizadas.

### Fase 2: Monitoreo de Bosques y Carbono
- [ ] **Superficie Bosque Natural**: Gráficos de cobertura boscosa vs no bosque.
- [ ] **Cambio Superficie**: Análisis de pérdida/ganancia de bosque.
- [ ] **Indicadores de Bosque**: Dashboard de métricas clave.
- [ ] **Inventario Forestal**: Visualización de datos de inventario.

### Fase 3: Sistema de Información Ambiental
- [ ] **Superficie Cubierta (Región)**: Mapa regional.
- [ ] **Autoridades Ambientales**: Directorio geoespacial.
- [ ] **Cambio Histórico**: Línea de tiempo de cambios ambientales.

### Fase 4: SNIFFS (Sistema Nacional)
- [ ] **Aprovechamientos**: Mapa de permisos y autorizaciones.
- [ ] **Movilizaciones**: Flujos de transporte de madera.
- [ ] **Decomisos**: Mapa de calor de incautaciones.
- [ ] **Plantaciones**: Registro nacional de plantaciones.
- [ ] **Registros**: Viveros y Regentes forestales.

### Fase 5: Información Comercial y Otros
- [ ] **Comercial**: Estudios de mercado, estadísticas de producción y boletines.
- [ ] **Otros**: Plantaciones registradas, Tierras PFC, Mapa Bosque, Sistemas Inversionista.

## Próximos Pasos Inmediatos
1.  Comenzar con la **Fase 1 (Planificación)**, específicamente con `FronteraAgricola.tsx`.
2.  Definir la estructura de datos (JSON/CSV) necesaria para alimentar los mapas y gráficos de esta sección.
