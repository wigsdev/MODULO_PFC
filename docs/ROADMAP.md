# Roadmap del Proyecto: Observatorio de Plantaciones Forestales Comerciales

Este documento describe la hoja de ruta estratégica para el desarrollo del Observatorio PFC.
**Última actualización:** 2024-12-16

## Fase 1: MVP y Estructura Base ✅ (Completada - Q4 2024)
- [x] Configuración inicial (Vite, React, TypeScript, Tailwind).
- [x] Arquitectura de componentes (Layouts, Atomos, Moleculas).
- [x] Implementación del Geovisor (Integración básica).
- [x] Dashboards Principales: Sector, Economía, Espacial, Normativa.
- [x] Migración de contenido estático a componentes dinámicos.
- [x] Despliegue inicial en GitHub Pages.

## Fase 2: Consolidación de Información ✅ (Completada - Dic 2024)
- [x] Dashboard Sector: 8 páginas con visualizaciones Recharts.
- [x] Dashboard Normativo: 4 secciones (Planificación, Lineamientos, Manejo, Incentivos).
- [x] Dashboard Espacial: 34+ páginas organizadas en subsecciones.
- [x] Integración de archivos JSON para datos dinámicos (28 archivos).
- [x] Gráficos interactivos con Recharts (BarChart, LineChart, PieChart).

## Fase 3: Expansión Economía (En Progreso - Q1 2025)
- [x] Proveedores de Bienes y Servicios.
- [x] Especies Aprovechadas.
- [x] Precios de Madera.
- [x] Transformación Industrial (CTP).
- [ ] Balanza Comercial.
- [ ] Exportaciones e Importaciones.
- [ ] Concesiones.
- [ ] Oferta Forestal y Volumen Movilizado.
- [ ] Costos de Plantaciones.

## Fase 4: Optimización y Calidad (Q1-Q2 2025)
- [ ] Optimización de carga (Lazy loading, optimización de imágenes).
- [ ] Resolución de warnings de Recharts (dimensiones -1).
- [ ] Auditoría de accesibilidad (WCAG).
- [ ] Mejoras de responsividad (Mobile/Tablet).
- [ ] Pruebas automatizadas (E2E con Playwright).

## Fase 5: Integración y Escalamiento (Q2 2025)
- [ ] Conexión con APIs en tiempo real (si aplica).
- [ ] Módulo de administración de contenidos (CMS/Admin Panel).
- [ ] Autenticación de usuarios (si se requiere acceso restringido).
- [ ] SEO y meta tags dinámicos.

## Fase 6: Lanzamiento y Mantenimiento
- [ ] Documentación final de usuario.
- [ ] Capacitación a administradores.
- [ ] Paso a producción en dominio final.
- [ ] Monitoreo de errores (Sentry o similar).
