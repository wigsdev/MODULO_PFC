# Roadmap del Proyecto: Observatorio de Plantaciones Forestales Comerciales

Este documento describe la hoja de ruta estratégica para el desarrollo del Observatorio PFC.

## Fase 1: MVP y Estructura Base (Q4 2024)
- [x] Configuración inicial (Vite, React, TypeScript, Tailwind).
- [x] Arquitectura de componentes (Layouts, Atomos, Moleculas).
- [x] Implementación del Geovisor (Integración básica).
- [x] Dashboards Principales: Sector, Economía y Espacial.
- [x] Migración de contenido estático a componentes dinámicos.
- [x] Despliegue inicial en GitHub Pages.

## Fase 2: Consolidación de Información y Normativa (Actual)
- [ ] Migración completa de datos desde documentos base.
- [ ] Implementación del Dashboard Normativo (Leyes, Reglamentos, Incentivos).
- [ ] Integración de visualización de datos (Gráficos Recharts/Chart.js).
- [ ] Secciones detalladas de Economía (Balanza comercial, Precios).

## Fase 3: Interactividad y Optimización (Q1 2025)
- [ ] Buscadores avanzados de normativa y especies.
- [ ] Filtros dinámicos en los mapas del Geovisor.
- [ ] Optimización de carga (Lazy loading, optimización de imágenes).
- [ ] Auditoría de accesibilidad (WCAG) y SEO.

## Fase 4: Integración y Escalamiento (Q2 2025)
- [ ] Conexión con APIs en tiempo real (si aplica).
- [ ] Módulo de administración de contenidos (CMS/Admin Panel).
- [ ] Autenticación de usuarios (si se requiere acceso restringido).
- [ ] Pruebas automatizadas (E2E con Cypress/Playwright).

## Fase 5: Lanzamiento y Mantenimiento
- [ ] Documentación final de usuario.
- [ ] Capacitación a administradores.
- [ ] Paso a producción en dominio final.
